# What you'll need

1. Openjdk 11 (https://openjdk.java.net/projects/jdk/11/)
2. Gradle 6.5.1 (https://gradle.org/install/)

# Development

## Building

```
$ gradlew clean build
```
## Testing

```
$ gradlew clean test
```
